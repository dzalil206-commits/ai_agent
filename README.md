# 🤖 Бот-ассистент по рассылкам

Этап 1: каркас бота с активацией по ОДНОРАЗОВЫМ кодам доступа.
ИИ и движок рассылок — на следующих этапах.

## ⚙️ Перед запуском
1. Открой `.env`, замени `ВСТАВЬ_СЮДА_ТОКЕН_ОТ_BOTFATHER` на токен от @BotFather.
2. База с 5000 кодов (bot.db) уже внутри — менять не нужно.

## 🔑 Коды доступа
- 5000 одноразовых кодов формата `WRN-XXX-XXX-XXX-XXX`.
- Список кодов — в файле `access_codes.txt` (отдаётся отдельно, ХРАНИ В СЕКРЕТЕ).
- Каждый код срабатывает один раз и привязывается к Telegram-аккаунту юзера.
- Тот же юзер заходит повторно без кода. Чужой аккаунт код не примет.

### Сгенерировать новые коды (если нужно)
```
python generate_codes.py 5000
```
⚠️ Перегенерация создаёт НОВЫЕ коды; старый список станет неактуальным.

## 📁 Структура (bot.py РЯДОМ с папками handlers и models!)
```
bot.py
config.py
texts.py            ← тексты бота
generate_codes.py   ← генератор кодов
requirements.txt
runtime.txt         ← версия Python (3.11) для хостинга
.env                ← сюда токен
bot.db              ← база с кодами (создаётся/обновляется генератором)
handlers/commands.py
models/database.py
```
Если папки handlers/ и models/ не рядом с bot.py → ошибка `No module named 'handlers'`.

## 🚀 Запуск
- Хостинг (BotHost и т.п.): залей все файлы с папками, пропиши BOT_TOKEN, старт `python bot.py`.
- Компьютер: `pip install -r requirements.txt` затем `python bot.py`.

## ✅ Проверка
1. Напиши боту → попросит код.
2. Введи любой код из access_codes.txt → "Код принят ✅".
3. Команды: /help, /stats, /start рассылки WB 200
